Este repositorio contiene una serie de proyectos incrementales diseñados para consolidar habilidades de ingeniería de software a nivel profesional utilizando el ecosistema moderno de Java. El roadmap está diseñado para dominar primero el núcleo del lenguaje antes de adoptar frameworks.

## 🧠 Filosofía de Desarrollo

- **Clean Code & Solid:** Código escrito para ser leído por humanos.
- **SaaS First:** Enfoque en soluciones multi-tenant.
- **Modern Java:** Uso intensivo de características de Java 21.

## 🚀 Ejercicios y Retos: Fase Core Java

### 1. Data Processing Engine (Java 21 Core)
Motor de análisis de datos en memoria para procesamiento masivo de logs/tickets utilizando concurrencia moderna.
- **Stack:** Java 21 (Virtual Threads, Stream API, Pattern Matching).
- [Ver Detalles](./reto-01-data-engine/README.md)

### 2. Solar Metrics Aggregator
Procesamiento de telemetría de hardware. Cálculo seguro de métricas utilizando tipos de alta precisión.
- **Stack:** Java Core (BigDecimal, Arrays, Math logic).
- [Ver Detalles](./reto-02-solar-metrics/README.md)


### 3. Access Control Validator
Motor de reglas basado en memoria para validación de permisos espaciales y temporales.
- **Stack:** Java Core (Collections Framework: Sets, Maps, Lists).

### 4. Chat Log Sanitizer
Pipeline de limpieza de datos no estructurados para extracción de entidades de negocio.
- **Stack:** Java Core (String manipulation, Regex, Text Blocks).

### 5. Multi-Tenant Data Partitioner
Motor de partición de datos para arquitecturas SaaS.
- **Stack:** Java Core (Advanced Stream API, Lambdas, Functional Interfaces).

### 6. Payment Strategy Simulator
Diseño de software puro aplicando principios SOLID para pasarelas de pago.
- **Stack:** Java Core (OOP, Interfaces, Polymorphism, Design Patterns).

## 🌐 Ejercicios y Retos: Fase Ecosistema Spring

### 7. Resilient RESTful API (Spring Boot 3)
Microservicio core con manejo avanzado de excepciones y validaciones estrictas.
